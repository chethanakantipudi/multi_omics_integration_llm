# Multi-Omics Integration Project

This repository contains the final Phase 2 project deliverables, including the continuous analysis flow from Phase 1 into Phase 2.

## Included files

- `phase1.ipynb` — Phase 1 notebook for eigengene preprocessing, TCGA/METABRIC integration, and initial analysis.
- `phase_2.ipynb` — Phase 2 continuation notebook that extends the project with survival alignment, model validation, embedding similarity analysis, and LLM interpretation.
- `Report_Phase 2 (1).pdf` — Final project report documenting the problem statement, data workflow, model findings, and conclusions.

## Notes

- `phase_2.ipynb` is intentionally included because it is a direct continuation of `phase1.ipynb`.
- The two notebooks together represent the full workflow for the multi-omics integration and survival analysis project.
- Duplicate or intermediate report files were removed so the repository only keeps the core deliverables.
